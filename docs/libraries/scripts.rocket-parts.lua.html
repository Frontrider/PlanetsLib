Public.on_built_rocket_silo (event)
On building a rocket silo, if the silo is vanilla-like, its recipe is replaced if it was placed on a surface that according to "Planetslib-planet-rocket-part-recipe"(mod-data), should be replaced. Forked from NotNotMelon's rocket silo code for Maraxsis.
